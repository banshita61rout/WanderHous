🏡 WanderHous

🚀 WanderHous — Full Stack Web Application
A fully functional and responsive full-stack web application designed to help users explore, discover, and list unique stays across different destinations.
WanderHous provides a seamless experience where users can browse listings, search locations, and become hosts to share their own properties. The platform focuses on clean UI, structured backend logic, and real-world application flow.

🌐 Live Demo

The project is deployed here:
(https://wanderhous.onrender.com/listings)

⚡ Quick Start — How to Run on Your Device

Follow these steps to run the project locally.

✅ Works on Windows, Mac, and Linux
⏱️ Setup time: 2–5 minutes

🔧 1. Clone the Repository
```
git clone https://github.com/banshita61rout/WanderHous.git
📁 2. Navigate to the Project Folder
cd WanderHous
📦 3. Install Dependencies
npm install
▶️ 4. Start the Server
npm start

After this, open in browser:
http://localhost:3000
```

🔗 System Flow (Frontend ↔ Backend ↔ Database)

This project follows a complete full-stack connection:

• The Frontend (EJS + CSS + JS) handles UI and user interaction
• The Backend (Node.js + Express) processes requests and manages routes
• The Database (MongoDB) stores listings, user data, and images Flow:

User → Frontend → Backend → Database → Backend → Frontend → User

🔗 This ensures:

• Dynamic content rendering
• Smooth user experience
• Real-time data handling
